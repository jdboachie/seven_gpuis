# seven_gpuis

Overview
--------
[`7GUIs`](https://eugenkiss.github.io/7guis/tasks/) is a small GUI programming benchmark consisting of seven tasks that exercise common GUI features. This repository aims to implement that benchmark using the `gpui` GUI library by [Zed Industries](https://zed.dev).
The name "seven gpuis" is a play on the original `7GUIs` benchmark and the `gpui` library used here. Each example is implemented with `gpui`, so it's literally "seven gpuis" ðŸ˜…
